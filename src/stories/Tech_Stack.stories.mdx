## Overview

This project is built using the following technologies:

- Framework - [Greenwood](https://www.greenwoodjs.io/) + [WCC](https://github.com/ProjectEvergreen/wcc) (Web Components)
- Styling - [Tailwind](https://tailwindcss.com/)
- Testing - [Web Test Runner](https://modern-web.dev/docs/test-runner/overview/)
- Linting - [ESLint](https://eslint.org/) and [Stylelint](https://stylelint.io/)

## Integrations

### EventBrite

[EventBrite](https://www.eventbrite.com/platform/) is used on the [Tickets page](https://www.blissfestri.com/tickets/) to provide an [embedded checkout experience](https://www.eventbrite.com/support/articles/en_US/Multi_Group_How_To/how-to-sell-eventbrite-tickets-on-your-website-through-an-embedded-checkout?lg=en_US).

Unless you run `localhost` on HTTPS, the full checkout experience will not be available locally, however any changes can be vetted by opening a PR and looking at the Netlify deploy preview.
